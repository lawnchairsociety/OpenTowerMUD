# Data Files

Configuration files for the MUD server.

## Files

- `city_rooms.yaml` - 10-room walled city layout (ground floor)
- `items.yaml` - Item definitions
- `npcs.yaml` - Town NPC definitions (non-hostile, 16 NPCs)
- `mobs.yaml` - Monster definitions (hostile)
- `spells.yaml` - Spell definitions
- `chat_filter.yaml` - Chat filter configuration
- `logging.yaml` - Logging configuration

## Runtime Files

- `tower.yaml` - Generated tower state (created at runtime)
- `opentowermud.db` - SQLite player database (created at runtime)
